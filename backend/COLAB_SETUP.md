# Colab Setup – Standard Operating Procedure (SOP)

## 1. Install Extension
Install the **"Google Colab"** extension for VS Code.

## 2. Cloud Storage Setup
- Go to **Google Drive**.
- Create a folder named **`football_cv_project`**.
- Upload your **`data.yaml`** file and **`datasets/`** folder into this new folder.

## 3. Connection
- Open **`backend/notebooks/train_colab.ipynb`**.
- Click **Select Kernel** (Top Right) → **Google Colab**.
- Sign in.

## 4. Execution
Run the cells **sequentially**. Output will stream directly to VS Code.
