# Spiideo `soccersegcal` (auxiliary)

Clone or copy the Spiideo soccersegcal repository here so it matches the layout expected by
`scripts.auxiliary_tools.generate_calibration` (root contains `soccersegcal/` and `sncalib/`).

Alternatively set `SPIIDEO_SEGCAL_ROOT` to an existing checkout path.

Production E2E homographies use vendored SoccerNet under `sn_calibration_vendor/`, not this tree.
